Contributors: Kevin Sylvestre
Tags: javascript, images, lightbox, slimbox
Requires at least: 2.7
Tested up to: 3.5
Stable tag: 1.0.7

Enables slimbox 2.03 on all image links including BMP, GIF, JPG, JPEG, and PNG links.

== Description ==

This plugin uses the jquery implementation of slimbox and makes use of [attr] style selectors by adding a section to the wordpress header.

For more information and examples of slimbox visit:

[Slimbox](http://www.digitalia.be/software/slimbox2)

== Installation ==

1. Upload `slimbox` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
